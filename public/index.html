Northcoders News API Endpoints

This API provides access to Northcoders News articles, topics, users, and comments.

GET /api

Serves a JSON representation of all available API endpoints.

GET /api/topics

Returns a list of available topics.

Example Response

{
  "topics": [
    { "slug": "mitch", "description": "The man, the Mitch, the legend" },
    { "slug": "cats", "description": "Not dogs" },
    { "slug": "paper", "description": "what books are made of" }
  ]
}
GET /api/users

Returns a list of users.

Example Response

{
  "users": [
    {
      "username": "lurker",
      "name": "do_nothing",
      "avatar_url": "https://www.golenbock.com/wp-content/uploads/2015/01/placeholder-user.png"
    }
  ]
}
GET /api/articles

Returns a list of articles. Supports filtering and sorting.

Queries

  • sort_by: Sorts by a valid column (defaults to `created_at`).
  • order: asc or desc (defaults to desc).
  • topic: Filters by topic slug.

Example Request

GET /api/articles?topic=mitch&sort_by=votes&order=asc

Example Response

{
  "articles": [
    {
      "article_id": 3,
      "title": "Eight pug gifs that remind me of mitch",
      "topic": "mitch",
      "author": "icellusedkars",
      "created_at": "2020-11-03T09:12:00.000Z",
      "votes": 0,
      "article_img_url": "...",
      "comment_count": 2
    }
  ]
}
GET /api/articles/:article_id

Returns a single article by id, including a comment_count.

Example Request

GET /api/articles/1

Example Response

{
  "article": {
    "author": "butter_bridge",
    "title": "Living in the shadow of a great man",
    "article_id": 1,
    "body": "I find this existence challenging",
    "topic": "mitch",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 100,
    "article_img_url": "...",
    "comment_count": 11
  }
}
GET /api/articles/:article_id/comments

Returns comments for an article, with the most recent first.

Example Request

GET /api/articles/1/comments

Example Response

{
  "comments": [
    {
      "comment_id": 1,
      "votes": 16,
      "created_at": "2020-04-06T12:17:00.000Z",
      "author": "butter_bridge",
      "body": "I hate this song.",
      "article_id": 1
    }
  ]
}
POST /api/articles/:article_id/comments

Adds a comment to an article and returns the new comment.

Request Body

{
  "username": "butter_bridge",
  "body": "I love this article!"
}
PATCH /api/articles/:article_id

Increments or decrements the votes on an article.

Request Body

{
  "inc_votes": 2
}
DELETE /api/comments/:comment_id

Deletes a comment by id. Responds with status 204 and no content.
